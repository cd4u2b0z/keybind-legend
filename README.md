# 󰌌 keybind-legend

> Hyprland keybind legend overlay — summoned with `Super + F1`.

A self-updating HTML reference for all active Hyprland keybindings. Lives in `~/.cache/hypr/` by design — the legend is regenerated as keybinds evolve, keeping the cache as the source of truth.

## 󰐊 Usage

`Super + F1` opens the legend in your browser. That's it.

## 󰒓 How It Works

The HTML file is generated/updated by tooling that parses your Hyprland config, so it always reflects your current keybindings. The `.cache` location is intentional — this is a living document, not a static asset.
